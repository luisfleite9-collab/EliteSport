# ⚽ Elite Sport - Oficial

A **Elite Sport** é uma plataforma de e-commerce de alta performance focada na venda de chuteiras de marcas renomadas como Nike, Adidas e Puma. O projeto apresenta uma interface moderna com estética "Neon Night", proporcionando uma experiência de compra imersiva e direta.

---

## 🚀 Funcionalidades

O projeto contempla toda a jornada de compra do usuário, dividida nas seguintes seções:

* **Autenticação:** Sistemas de Login e Cadastro de novos membros com validação de campos (E-mail, CEP, Endereço).
* **Hero Section:** Apresentação de impacto com destaque para atletas e chamada para a loja.
* **Catálogo Dinâmico:** Grade de produtos com cards detalhando nome, preço promocional e botão de compra.
* **Carrinho de Compras:** Resumo do pedido com cálculo de subtotal e frete grátis.
* **Checkout Multicanal:** * **PIX:** Gerador de QR Code e função "Copia e Cola" para aprovação instantânea.
* **Cartão de Crédito:** Formulário completo com seleção de parcelamento em até 12x.


* **Confirmação de Pedido:** Tela de sucesso com geração de número de pedido exclusivo.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais da web:

* **HTML5:** Estruturação semântica de todas as seções.
* **CSS3:** Estilização personalizada, incluindo:
* Design responsivo (Mobile First).
* Tipografia via Google Fonts (Montserrat).
* Efeitos de brilho Neon e Transições suaves.
* Layouts em Grid e Flexbox.

## 🎨 Identidade Visual

A interface utiliza uma paleta de cores voltada para o público esportivo moderno:

* **Fundo:** `#000000` e `#111111` (Dark Mode).
* **Destaque (Neon):** `#7CFC00` (Verde Limão / Neon Green).
* **Tipografia:** Montserrat (Variações de 400 a 900 de peso).

---

## 🔧 Como Executar o Projeto

1. Clone este repositório ou baixe os arquivos.
2. Certifique-se de que a pasta `imagens/` contenha as fotos referenciadas no HTML.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
